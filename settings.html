Settings

This game is a 2D Game based on a Customizable Flappy Bird. This Game runs with a Software called Eblava which supports the maps, More maps are coming soon.
Each map has its own unique Theme and Speed for gameplay. Fun Fact: This game was a mess for the longest time until i remastered it

How to Update

You shouldnt have any problems while installing updates, You firstly have to go to the download website, Its located on the Main Menu
https://tinyurl.com/SquareFlapper Upon installing the Latest Version You can open it the same way you did or you can copy all the files and overrite them in your old games folder

Version
